---

## ✅ 사용 방법

매일 아래 프롬프트를 Claude에게 그대로 붙여넣어 사용하세요.
이전에 출제된 문제와 겹치지 않도록 회차를 직접 기입하거나, Claude가 알아서 관리합니다.

---

## 📌 기본 프롬프트 (매일 복사해서 사용)

```
오늘의 2026 노인맞춤돌봄서비스 사업안내 퀴즈를 출제해줘.

【출제 기준】
- 기본 문제(선택형) 5문항: 군별 서비스 시간, 인력 기준, 재사정 주기 등 핵심 수치·기준 중심
- 어려운 문제(선택형) 3문항: 조건 분기(협의체 심의 대상 vs 생략 등), 2025→2026 변경사항, 예외규정 등 혼동하기 쉬운 내용 중심
- 어려운 문제(서술형) 2문항: 실무 사례형(특정 상황에서의 절차·조치 서술), 근거 페이지 포함

【공통 규칙】
- 모든 문제는 반드시 2026 사업안내(프로젝트 지식)에서 근거를 확인하고 출제
- 정답과 함께 사업안내 페이지 번호 명시
- 이전 출제 문제와 최대한 중복 없이 새로운 주제로 출제
- 선택지는 실제 현장에서 혼동하기 쉬운 오답 포함

【출력 형식】
문제 번호 / 문제 / 선택지 / 정답 / 근거(사업안내 페이지) 순으로 출력
서술형은 문제 / 모범답안 / 근거 페이지 순으로 출력

퀴즈 종료 후, 오늘 문제를 .docx 파일로 문서화해줘.
```
