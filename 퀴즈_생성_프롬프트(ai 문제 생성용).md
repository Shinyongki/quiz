# 퀴즈 문제 생성 프롬프트

---

## 1회차 (최초 생성 시)

```
1회차 퀴즈 JSON 생성해줘.

session: 1
date: 오늘 날짜
```

---

## N회차 (누적 진행 시)

```
N회차 퀴즈 JSON 생성해줘.

session: N
date: 오늘 날짜
이전 회차 출제 범주: (예: A 1문항, B 2문항, C 1문항, D 3문항, E 1문항, H 2문항)
이번 회차 집중 범주: (없으면 생략 → 자동 혼합)
```

---

## 심화 회차 (Dashboard 분석 후)

```
N회차 심화 퀴즈 JSON 생성해줘.

session: N
date: 오늘 날짜
저장 문항 범주 분포: (예: D 8건, C 5건, H 4건) ← Dashboard 차트에서 확인
위 범주 집중 출제, 어려운 문제 비중 높여줘.
```

---

## 특정 범주 집중 시

```
N회차 퀴즈 JSON 생성해줘.

session: N
date: 오늘 날짜
집중 범주: F (퇴원환자 단기집중 서비스) 위주로 출제
```

---

## 범주 코드 참고

| 코드 | 범주명 |
|------|--------|
| A | 서비스 대상·자격 |
| B | 서비스 시간·내용 |
| C | 인력·급여·근무 |
| D | 재사정·심의 |
| E | 종결·사후관리 |
| F | 퇴원환자 단기집중 |
| G | 특화지원 |
| H | 2025→2026 변경사항 |
